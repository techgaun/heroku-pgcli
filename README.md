# Usage
<!-- usage -->
```sh-session
$ npm install -g heroku-pgcli
$ oclif-example COMMAND
running command...
$ oclif-example (-v|--version|version)
heroku-pgcli/0.1.0 linux-x64 node-v8.12.0
$ oclif-example --help [COMMAND]
USAGE
  $ oclif-example COMMAND
...
```
